Backend startet:
npm start

Ordnerstruktur:
- dao: Datenbankzugriff
- services: Logik, Validierung
- routes: API Endpoints
- db: Datenbankdateien

Server hört auf Port 8000


Frontend starten (Port 3000): 
cd frontend
npm install     # nur beim ersten Start notwendig
node server.js

Frontend läuft auf http://localhost:3000

Aufruf im Browser:
http://localhost:3000
